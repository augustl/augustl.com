date: 2010.10.27 title: Custom scrollbar, native behaviour rss_warning: true

Tldr: Use a hidden native scrollbar for scrolling, use JavaScript to read it's current position, move the custom scrollbar based on the position of the native scrollbar. Mouse wheel, page up/down, physical scrolling on Macs, etc, works just as expected.

Noscript clients gets a normal native overflow scroll.

Disclaimer: this particular custom scrollbar is ugly. The purpose of the demo is to show how it behaves when scrolling with mouse wheel, arrows, etc.
